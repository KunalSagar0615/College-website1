# College Management System - Frontend

A responsive and user-friendly college management frontend built using **React.js**, **Tailwind CSS**, and **Vite**. The project provides separate interfaces for students and administrators with features like authentication, profile management, notifications, gallery, enquiry, complaints, and admin dashboard integration.

## 🚀 Features

* Student & Admin Login/Register
* Student Profile Management
* Notifications Management
* Gallery Section
* Admission Enquiry Form
* Complaint & Contact Forms
* Responsive Admin Dashboard
* REST API Integration

## 🛠️ Tech Stack

* React.js
* Tailwind CSS
* JavaScript
* Vite
* REST APIs

## 📂 Project Setup

```bash
# Clone repository
git clone <your-repo-link>

# Navigate to project folder
cd College-website1

# Install dependencies
npm install

# Start development server
npm run dev
```
